# Arithmetic Operators

10 + 20                 # 30
20 - 10                 # 10
10 * 20                 # 200
20 / 10                 # 2

10 ** 2                 # 100

Automatic floating point conversion.

10.0 + (10 + 20)        # 40.0
20.0 - (10 + 10)        # 0.0
10.0 * (10 * 2)         # 200.0

Common division error.

30 / 20                 # 1
30.0 / 20.0             # 1.5

Integer divison, or quotient.

20.0 // 20.0            # 1.0
30.0 // 20.0            # 1.0
40.0 // 20.0            # 2.0

Modulo, or remainder.

12.5 % 10.0             # 2.5
10.0 % 20.0             # 10.0

Built in numerical operations.

abs(-20)                # 20
sum([1, 2, 3, 4])       # 10
min(1, 2, 3, 4)         # 1
max(1, 2, 3, 4)         # 4

Rounding values.

round(2.945)            # 3.0
round(2.495)            # 2.0

round(2.945, 2)         # 2.94
